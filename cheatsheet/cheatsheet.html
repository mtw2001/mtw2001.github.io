Michael Wentworth Projects About me Cheatsheet

This is where I will organize tips and tricks that I learn throughout the course!

HTML

Tip Content
Directory navigation To refer to files in an directory located above the current, use "..". Similarly, to
refer to files located two directories above, use "../.."
Linking to source files CSS: link rel="stylesheet" href="style.css", JavaScript: script src="index.js"
Element ID Assigns IDs to elements using "id=name". This allows you to refer to it from JS.
CSS within HTML CSS can be included directly in HTML using style tag

CSS

Tip Content
Hover You can change the state of an item when it is hovered over within the CSS file
by defining a specific hover state using ":hover"
Text align Align text to the center of the page using text-align: center;
Hierarchy Styling inherits from parent elements. For example, a header
that is within the body tag will inherit all body styling unless
it is overidden by header styling.

Javascript

Tip Content
Initializing variables early Initialize variables to be "= null" and then set values later
so that the variables can be accessed by any functions
JQuery JQuery references HTML elements using "$('#elementid')" followed
by whatever JQuery function you want.
Window controls window.location.href = to set the url of the page
or window.location.relead() to reload the page.
